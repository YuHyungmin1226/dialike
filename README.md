# diaLike - 디아블로 스타일 게임 시스템 웹 구현

**diaLike**는 디아블로(Diablo) 시리즈의 핵심 게임 디자인 요소를 분석하여 웹 브라우저 환경에 구현한 프로젝트입니다. 파밍(Looting), 아이템 무작위화(Randomization), 장비 슬롯 장착, 능력치 스케일링, 몬스터 비선형 성장 등 액션 RPG의 핵심 시스템을 순수 웹 기술로 구현했습니다.

## 주요 기능

- **장비 슬롯 시스템**: 무기(weapon), 방패(shield), 투구(helmet), 갑옷(chest) 4대 부위 독립 장착
- **무작위 접사(Affix) 시스템**: Prefix/Suffix 테이블 기반 무작위 옵션 부여로 Magic/Rare/Unique 등급 구현
- **아이템 레벨 스케일링**: 몬스터 레벨 기반 요구 레벨 동적 산출 및 능력치 비례 보정
- **레벨 제약 시스템**: 레벨 부족 시 장착 차단 및 붉은색 경고 UI
- **몬스터 비선형 성장**: 플레이어 성장 곡선에 대응하는 비선형 난이도 보정
- **실시간 전투**: 애니메이션 기반 전투 시스템

## 기술 스택

- HTML5, CSS3, JavaScript (Vanilla)
- Canvas 기반 렌더링
- 반응형 웹 디자인

## 시작하기

`index.html` 파일을 웹 브라우저에서 열어 바로 실행할 수 있습니다. 별도의 서버나 빌드 과정이 필요하지 않습니다.

## 프로젝트 구조

```
dialike/
├── index.html          # 메인 게임 페이지
├── style.css           # 스타일시트
├── js/                 # 게임 로직 JavaScript
├── assets/             # 이미지 및 리소스 파일
├── test/               # 테스트 파일
├── dia_task.md         # 디아블로 시스템 분석 및 구현 보고서
└── README.md           # 프로젝트 문서
```
